Installation Instructions

Windows:

  • 1. Download the Windows .zip file
  • 2. Extract the archive to a folder of your choice
  • 3. Run PinoySkater.exe
  • 4. Enjoy the game!

Linux:

  • 1. Download the Linux .zip file
  • 2. Extract the archive: unzip PinoySkater-Linux.zip
  • 3. Navigate to the folder: cd PinoySkater
  • 4. Make executable: chmod +x PinoySkater
  • 5. Run: ./PinoySkater
